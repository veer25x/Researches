# Heart Disease Prediction using Machine Learning

This project predicts the presence of heart disease using machine learning algorithms.

## Dataset
UCI Heart Disease dataset (Kaggle)

## Algorithms Used
- Logistic Regression
- Random Forest

## Accuracy
- Logistic Regression: 78.8%
- Random Forest: 83.15%

## Tools
Python, Pandas, Scikit-learn, Matplotlib

## Conclusion
Random Forest performed better and can assist in early heart disease detection.
